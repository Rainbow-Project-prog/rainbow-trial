Rainbow Trial — 効果音ファイル配置ガイド
==========================================

このディレクトリに以下のファイルを配置してください。
ファイルが存在しない場合、アプリはエラーを出さず無音で動作します。

■ 必要なファイル一覧
----------------------------------------
signal-normal.mp3       シグナル受信音(NORMAL / GOOD)
signal-rare.mp3         シグナル受信音(RARE / EPIC)
signal-legendary.mp3    シグナル受信音(LEGENDARY 専用)
tp-hit.mp3              TP HIT (勝利)
sl-hit.mp3              SL HIT (敗北)
level-up.mp3            レベルアップ
achievement.mp3         実績解除
button-tap.mp3          ボタン押下
mission-complete.mp3    ミッション達成

■ 推奨フォーマット
----------------------------------------
- 形式: MP3 (128kbps 以上推奨)
- 長さ: 0.3〜3秒程度
- 音量: ノーマライズ済みのもの

■ 無料音源サイト
----------------------------------------
- 効果音ラボ        https://soundeffect-lab.info/
- Pixabay           https://pixabay.com/sound-effects/
- freesound.org     https://freesound.org/
- 魔王魂            https://maou.audio/

■ 音源のイメージ
----------------------------------------
signal-normal.mp3       : 軽い「ポーン」通知音
signal-rare.mp3         : 神秘的な「キラキラ」音
signal-legendary.mp3    : ファンファーレ風の派手な音
tp-hit.mp3              : 「ジャーン」勝利音・明るいSE
sl-hit.mp3              : 控えめな「ガクン」落下音
level-up.mp3            : ゲーム風ファンファーレ
achievement.mp3         : バッジ獲得の「チャリン」音
button-tap.mp3          : 軽い「タップ」音
mission-complete.mp3    : チェック完了の「ピン」音
